# AI Integration Capstone Project — Static Website

This is a pure static website version of the capstone project.

## Deploy to Vercel

1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. Import the repository into Vercel.
4. Framework preset: **Other**.
5. Build command: **leave blank**.
6. Output directory: **leave blank**.
7. Deploy.

## Deploy to GitHub Pages

1. Push this folder to a GitHub repository.
2. In GitHub, open **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the default branch and `/root`.
5. Save.

## Local Preview

Because this is static HTML/CSS/JS, you can preview with any simple local web server.

Example with Python:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.
